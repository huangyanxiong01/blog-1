# 正则表达式
  - 是什么：用特定的符号表示一类字符串的集合
  - 怎么做
    - 第一步：举例 + 分解 + 每部分用正则描述 => 得出雏形
    - 第二步：扩展举例 + 检查每一部分的正则描述，若集合范围大了则增加正则符号，若集合范围小了则减少正则符号 => 得到完善的表达式
    - 重复第二步，直到正确为止
  - 正则表达式的组件
    - 普通字符
      - 数字
      - 字母
      - 标点符号
      - 其他符号
    - 非打印字符
      - \cx：匹配 Control-x 类型的键盘输入，比如 \cC 代表 Control-C 中断信号的输入
      - \s：匹配空白字符，比如换行符、回车符、制表符等等
      - \S：匹配非空白字符，\s 的取反
