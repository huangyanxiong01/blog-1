# 方法

## 描述

为了突出重点，这个例子必须足够简单而且完整，这里就以投票功能为例子。

- 需求描述：一个投票系统，假设任何人都可以对每一位候选人进行投票，系统统计每个候选人的得票数，并显示出来
- 交互描述：有两个页面，一个是投票页面，包含单选框的表单；另一个是候选人得票数页面，`<ul>` 和 `<li>` 组成的列表。默认显示候选人得票数页面，点击投票按钮跳转投票页面进行投票，投票完成提交表单，跳转回得票数页面
