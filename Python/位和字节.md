## 位运算

例子：

a = 0011 0101
b = 1010 0011

### 与运算

两个运算位都是 1 时，结果是 1，否则是 0。

a & b = 0010 0001

### 或运算

两个运算位都是 0 时，结果是 0，否则是 1。

a | b = 1011 0111

### 异或运算

两个运算位不同时，结果是 1，否则是 0。

a ^ b = 1001 0110

### 非运算

取反

~a = 1100 1010

## 字节数组

bitarray (参考：https://pypi.org/project/bitarray/)

例子：

```
from bitarray import bitarray
bit_array = bitarray(10)
bit_array.setall(0)
bit_array[3] = 1
```
